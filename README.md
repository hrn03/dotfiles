# dotfiles

My configuration files for MacOS & zsh.

## Features

### `peeking_cat`

When the terminal starts, a cat peeks out from the bottom of the screen to greet you.

## Recommended Commands

```bash
sudo port selfupdate
sudo port install fzf bat eza
```

## Environments

The configurations in this repository are built and tested under the following environment:

- OS: macOS
- Shell: zsh
- Terminal Emulator: kitty
- Editor: Neovim
- Prompt: Powerlevel10k
